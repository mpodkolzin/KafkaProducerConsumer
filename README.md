### Basic Kafka Consumer/Producer utility

#### How to use

##### 1. From command line
* Clone solution, and build it.
KafkaProducerConsumer\KafkaConsumer\bin\Debug>KafkaConsumer.exe --brokers kafka-001:9092 --topic test_topic

##### 1. From Visual Studio

* Clone solution, and build it.
In KafkaConsumer project properties -> Debug -> Command Line arguments add
 --brokers kafka-001:9092 --topic test_topic
